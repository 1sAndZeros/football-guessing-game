FOOTBALL GUESSING GAME

EACH GUESS GIVES CLUES TO THE ACTUAL PLAYER E.G. TEAM, LEAGUE, AGE, NATIONALITY, POSITION.

API USED IS FOOTBALL-API FROM RAPID API
